# Projeto QGIS

O projeto cartográfico será mantido em SIRGAS 2000 / UTM zona 23S (`EPSG:31983`).

Camadas mínimas previstas:

- limite do município;
- distritos e subprefeituras;
- polígonos ou centroides dos cemitérios;
- ponto do crematório;
- pontos das agências funerárias;
- categorias tarifárias;
- destinos da gratuidade;
- indicadores socioeconômicos selecionados.

Os arquivos de projeto não deverão depender de caminhos absolutos da máquina do pesquisador. Estilos serão armazenados separadamente e as fontes das camadas externas serão registradas em documentação.

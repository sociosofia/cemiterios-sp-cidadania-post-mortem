# Mapas

Esta pasta receberá apenas mapas derivados de bases conferidas.

Produtos previstos:

- localização dos cemitérios por categoria tarifária;
- destinos dos sepultamentos gratuitos;
- divisão dos equipamentos por bloco e concessionária;
- distribuição das agências funerárias;
- comparação entre categoria tarifária, centralidade e indicadores socioeconômicos;
- acessibilidade e distância entre distritos de residência e destinos funerários.

Todo mapa deverá apresentar fonte, data de referência, projeção cartográfica e aviso quando uma camada ainda for provisória.

# Dados brutos

Esta pasta registra o inventário das fontes originais e, quando juridicamente adequado, arquivos públicos necessários à reprodução.

Regras:

- preservar nome, data de acesso e endereço da fonte;
- não editar arquivos brutos;
- evitar duplicar documentos integrais quando um endereço oficial estável for suficiente;
- não publicar dados pessoais, entrevistas, memórias ou anotações privadas;
- acompanhar cada conjunto de dados de metadados sobre origem, vigência e licença.

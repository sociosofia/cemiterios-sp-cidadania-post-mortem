# Validação cartográfica - 20/07/2026

Esta branch executa o pipeline completo para validar:

- geocodificação das 40 agências funerárias;
- restrição dos pontos ao limite municipal;
- geometria oficial dos cemitérios;
- geração do mapa real em PNG e SVG;
- publicação de artefato para inspeção antes da incorporação definitiva.

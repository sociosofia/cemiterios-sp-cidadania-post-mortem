# Dados processados

Produtos derivados e auditáveis da pesquisa.

Formatos previstos:

- `equipamentos.csv` — tabela principal;
- `cemiterios.geojson` — pontos ou centroides conferidos;
- `agencias.geojson` — pontos das agências funerárias;
- `cemiterios.gpkg` — camadas em SIRGAS 2000 / UTM 23S;
- tabelas de tarifas com datas de início e fim de vigência.

Todo arquivo deverá ser gerado por rotina documentada ou acompanhado de registro das alterações manuais realizadas.
